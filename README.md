# AI-_STUDY_PLANNER
Smart, adaptive study planning system
Project: AI Study Planner
Description: A smart, adaptive study planning system that generates personalized study schedules based on subject difficulty, deadlines, learning pace, and cognitive load.

How to Run:
1. Install required packages: pip install -r requirements.txt
2. Run the app: streamlit run enhanced_study_planner.py
3. Explore features:
   - Add subjects & difficulty
   - Set deadlines & learning pace
   - Generate weekly timetable
   - View subject priority & weak areas

Screenshots are included in the assets folder.
